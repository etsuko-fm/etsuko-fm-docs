# Panning
<div className="content-container">
![screenshot of panning page](https://d21ikh70ytbexu.cloudfront.net/docs/eterna/260124/panning.png)

Controls the panning of all voices. The position of the voices in the stereo field is spread within the available width (E3), and can be continuously cycled using twist (E2).



#### Controls
| Control | Function | Description |
|:--------|:---------|:------------|
| K2 | Toggle LFO on/off | Enables or disables the LFO |
| K3 | Select LFO waveform | Chooses the waveform shape for the LFO |
| E2 | Twist / LFO rate | Controls the panning of all voices at the same time; or when LFO is on, LFO rate |
| E3 | Width | Maximum stereo width |
